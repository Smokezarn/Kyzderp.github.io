Text Format to JSON Text Parser

Input text like how you would normally type it in chat, click Parse and it will generate JSON text that you can use, for example in a command block with /tellraw @a [{text:"Hi!",color:dark_green}]
Does not support the "reset" format (&r), and changing color immediately after changing special format still keeps the format (i.e. &a&lHi and &l&aHi will show the same bold green).
